📌 AEO（Answer Engine Optimization — 答案引擎優化）
評估你的內容在 AI 答案引擎中的表現。涵蓋：
• EEAT（專業性、權威性、可信度）— 權重 40%
  檢測作者資訊、引用來源、結構化資料標記、信任信號、內容深度
• 相關性（Relevance）— 權重 30%
  分析標題品質、meta 描述、標題結構、內容深度、圖片使用
• 結構（Structure）— 權重 20%
  檢查標題層級、列表、表格、段落組織
• 時效性（Freshness）— 權重 10%
  根據內容更新時間進行衰減評分（30天內=滿分，超過180天大幅衰減）
• LLM 語氣分析 — 使用多語言 BERT 本地模型（五星情緒）映射為語氣分數
• 獨特性檢測 — 使用 MiniLM 嵌入模型評估內容獨特性

📌 SEO（Search Engine Optimization — 搜尋引擎優化）
評估你的網頁在傳統搜尋引擎中的表現。涵蓋：
• 技術 SEO（Technical）— 權重 30%
  檢測 canonical URL、viewport、HTTPS、語義化 HTML、H1 標籤
• 頁面 SEO（On-Page）— 權重 35%
  分析標題長度、描述長度、標題數量、字數、首標題結構
• 連結配置（Link Profile）— 權重 20%
  評估內部連結、外部連結數量及平衡性
• 圖片 SEO（Image SEO）— 權重 15%
  檢查 alt 文字覆蓋率、圖片數量

📌 GEO（Generative Engine Optimization — 生成式引擎優化）
評估你的內容被 AI 生成式引擎（如 Google AI Overview、ChatGPT 搜尋）引用和推薦的能力。涵蓋：
• 可引用性（Citability）— 權重 30%
  檢測引用數量、統計數據密度、內聯引用、外部權威連結
• 事實密度（Factual Density）— 權重 25%
  分析每千字統計數據量、引用密度、引述使用、內容深度
• 結構化答案（Structured Answers）— 權重 25%
  檢測 FAQ 區塊、比較表格、步驟指南、答案膠囊
• 權威性（Authority）— 權重 20%
  評估作者標記、結構化資料、更新時間、外部連結、內容長度

📌 LLMO（LLM Optimization — 大型語言模型優化）
評估你的內容對大型語言模型的友善程度。涵蓋：
• 可爬取性（Crawlability）— 權重 20%
  檢測語義化 HTML 使用、行銷語言密度、段落長度適中性
• 完整性（Completeness）— 權重 30%
  評估字數、標題覆蓋、段落數量、列表使用、表格使用
• 直接回答（Direct Answers）— 權重 30%
  檢測定義模式、比較信號、步驟指南、比較表格、FAQ
• 清晰度（Clarity）— 權重 20%
  分析摘要區塊、關鍵概念加粗、標題層級、語言精煉度

✅ 100% 本地運行，隱私絕對安全
所有分析都在你的瀏覽器中完成。你的網頁內容、分析結果、LLM 推理——全部不會離開你的電腦。不需要 API Key、不需要帳號、不需要網路連線（模型已內建）。在當今隱私意識抬頭的時代，這是最安心的選擇。

✅ 四大評分一次到位，省去多個工具切換
市面上有專門的 SEO 工具、GEO 工具、LLM 分析工具，但沒有一個工具能同時涵蓋這四個維度。AEO Score Calculator 一鍵完成全部分析，讓你對網頁的 AI 時代優化程度有全景式的了解。

✅ 本地 LLM 推理，無需付費 API
內建多語言 BERT（語氣用五星情緒）和 MiniLM（文本嵌入）模型，直接在瀏覽器中運行。你不需要支付 OpenAI、Anthropic 或任何第三方 API 的費用，也不需要擔心 API 限流或服務中斷。

✅ 具體可行的改善建議，而非空泛評分
每個評分維度都會生成具體的改善建議，按嚴重程度分為：
• 🔴 Critical（嚴重）— 必須立即處理的問題
• 🟡 Warning（警告）— 影響評分的重要因素
• 🔵 Info（建議）— 進一步優化的方向
每條建議都包含具體的行動指引，例如「添加 FAQPage 結構化資料」、「在 H2 標題後加入 40-60 字的直接回答」等。

✅ 匯出 AI 友善的 Markdown 報告
一鍵複製或下載 Markdown 格式的完整分析報告，包含 URL、時間戳、各維度評分、元數據和所有改善建議。你可以直接將報告貼到 ChatGPT 或 Claude 中，讓 AI 為你生成具體的程式碼修改建議。

✅ 離線可用，不受網路限制
所有模型（約 188MB，含打包後 dist/）內建在擴充功能中，安裝後即使完全離線也能正常運行。適合在飛機上、會議中或網路不穩定的環境下使用。
